Alt + G: 模型居中

Alt + z: 进入 X-Ray 模式。所有模型半透明
z 选择预览模式
  - 线框
	- 实体
		- 视图着色方式(右上角下拉)-颜色-随机

Num 1 Num6 Num6 Num8 Num8
Ctrl + Alt + Num0 摄像机视角

G + Z, Ctrl + 滑动鼠标

编辑模式(Tab)
	- 点模式 Ctrl + 1
	- 线模式 Ctrl + 2
	- 面模式 Ctrl + 3

选择面
	i 收缩面
	e 挤压

独显模式 / 

倒角: 有的面大有的面小，模型拉伸(S)需要应用拉伸

Ctrl + 左键：分块
Ctrl + 滚轮 + 左键: 多级分块
