基础货币: 实物货币
派生货币: (债务累积) 银行贷款利息获利

``` 
- 存款准备金率: 20%
- Bank A 拥有金融牌照

Wang  ---> $100  ---> Bank A

Li    <--- $80   <---  $20($100 * 20%)
      ---> $80   --->  Bank A

Zhang <--- $64   <---  $16($80 * 20%)
      ---> $64   --->  Bank A
                  
Liu   <--- $51.2 <---  $12.8($64 * 20%)
      ---> $51.2 --->  Bank A

      ....
```

基础货币: $100
派生货币: $80 + $64 + $51.2 + ... = $400

货币乘数计算公式: k = (Rc + 1) / (Rd + Re + Rc)

- Rd: 法定准备金率
- Re: 超额准备金率
- Rc: 现金在存款中的比率

本例假定 Rc = 0, Rd + Re = 20%
k = (0 + 1) / (20% + 0) = 5

即基础货币 $100 最后的货币总量为 $100 * k = $500
